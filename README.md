# marvel-heroes
* Responsive Design
* Dark Mode
* Editable name and description
* Favorite & Blacklist
* Search hides blacklisted characters
* Vercel deploy: https://marvel-heroes-mu.vercel.app/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
